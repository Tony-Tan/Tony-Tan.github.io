Quadratic Functions[1]

Quadratic function, a type of performance index, is universal. One of its key properties is that it can be represented in a second-order Taylor series precisely.

F(x)=12xTAx+dx+c(1)F(\boldsymbol{x})=\frac{1}{2}\boldsymbol{x}^TA\boldsymbol{x}+\boldsymbol{d}\boldsymbol{x}+c\tag{1} F(x)=21​xTAx+dx+c(1)

where AAA is a symmetric matrix(if it is not symmetric, it can be easily converted into symmetric).
And recall the property of gradient:

∇(hTx)=∇(xTh)=h(2)\nabla (\boldsymbol{h}^T\boldsymbol{x})=\nabla (\boldsymbol{x}^T\boldsymbol{h})=\boldsymbol{h}\tag{2} ∇(hTx)=∇(xTh)=h(2)

and

∇(xTQx)=Qx+QTx=2Qx(3)\nabla (\boldsymbol{x}^TQ\boldsymbol{x})=Q\boldsymbol{x}+Q^T\boldsymbol{x}=2Q\boldsymbol{x}\tag{3} ∇(xTQx)=Qx+QTx=2Qx(3)

then the first-order of quadratic functions is:

∇F(x)=Ax+d(4)\nabla F(\boldsymbol{x})=A\boldsymbol{x}+\boldsymbol{d}\tag{4} ∇F(x)=Ax+d(4)

and second-order of quadratic functions is:

∇2F(x)=A(5)\nabla^2 F(\boldsymbol{x})=A\tag{5} ∇2F(x)=A(5)

More than second-order derivatives are 000.

The shape of quadratic functions can be described by eigenvalues and eigenvectors of its Hessian matrix. Hessian matrix is a symmetric matrix and its eigenvectors are mutually orthogonal, let:

z1,z2,⋯ ,zn(6)\boldsymbol{z}_1,\boldsymbol{z}_2,\cdots ,\boldsymbol{z}_n\tag{6} z1​,z2​,⋯,zn​(6)

denote the whole set of the eigenvectors of the Hessian matrix AAA and we build a matrix:

B=[z1z2⋯zn](7)B=\begin{bmatrix} \boldsymbol{z}_1&\boldsymbol{z}_2&\cdots &\boldsymbol{z}_n \end{bmatrix}\tag{7} B=[z1​​z2​​⋯​zn​​](7)

and then we have BBT=IBB^T=IBBT=I then B−1=BTB^{-1}=B^TB−1=BT. When Hessian matrix is positive definite, it would have nnn eigenvectors(Hessian has a full rank nnn) and we can change Hessian matrix into a new matrix whose basises are the set of eigenvectors:

A′=BTAB=[λ1λ2⋱λn]=Λ(8)A'=B^TAB=\begin{bmatrix} \lambda_1&&&\\ &\lambda_2&&\\ &&\ddots&\\ &&&\lambda_n \end{bmatrix}=\Lambda\tag{8} A′=BTAB=⎣⎢⎢⎡​λ1​​λ2​​⋱​λn​​⎦⎥⎥⎤​=Λ(8)

after changing the basis, the new Hessian matrix is a diagonal matrix whose elements on the diagonal are the eigenvalues. This process can be inversed because of BBT=IBB^T=IBBT=I:

A=BA′BT=BΛBT(9)A=BA'B^T=B\Lambda B^T\tag{9} A=BA′BT=BΛBT(9)

We can calculate the derivative towards any directions:

pT∇2F(x)p∣∣p∣∣2=pTAp∣∣p∣∣2(10)\frac{\boldsymbol{p}^T\nabla^2 F(\boldsymbol{x})\boldsymbol{p}}{||\boldsymbol{p}||^2}=\frac{\boldsymbol{p}^TA\boldsymbol{p}}{||\boldsymbol{p}||^2}\tag{10} ∣∣p∣∣2pT∇2F(x)p​=∣∣p∣∣2pTAp​(10)

For columns of BBB can span the whole space, So we can find a vector c\boldsymbol{c}c satisfy:

p=Bc(11)\boldsymbol{p}=B\boldsymbol{c}\tag{11} p=Bc(11)

then take equation(8), equation(11) into equation(10) we get:

cTBTABccTBTBc=cTΛccTIc=∑i=1nλici2∑i=1nci2(12)\frac{\boldsymbol{c}^TB^TAB\boldsymbol{c}}{\boldsymbol{c}^TB^TB\boldsymbol{c}}=\frac{\boldsymbol{c}^T\Lambda\boldsymbol{c}}{\boldsymbol{c}^TI\boldsymbol{c}}=\frac{\sum^n_{i=1}\lambda_i c_i^2}{\sum_{i=1}^n c^2_i}\tag{12} cTBTBccTBTABc​=cTIccTΛc​=∑i=1n​ci2​∑i=1n​λi​ci2​​(12)

From equation(12), we could conclude:

λmin≤pTAp∣∣p∣∣2≤λmax(13)\lambda_{\text{min}}\leq \frac{\boldsymbol{p}^TA\boldsymbol{p}}{||\boldsymbol{p}||^2}\leq \lambda_{\text{max}}\tag{13} λmin​≤∣∣p∣∣2pTAp​≤λmax​(13)

  1. maximum 2nd2^{\text{nd}}2nd derivative occure along with zmax\boldsymbol{z}_{\text{max}}zmax​(according to λmax\lambda_{\text{max}}λmax​)
  2. eigenvalues is the 2nd2^{\text{nd}}2nd derivative along its eigenvector.
  3. eigenvectors could define a new coordinate system
  4. eigenvectors are principal axes of the function contour.
  5. going along with the zmax\boldsymbol{z}_{\text{max}}zmax​ direction could have the largest change in function value ∣ΔF(x)∣|\Delta F(x)|∣ΔF(x)∣
  6. eigenvalues here are all positive because of positive definite.

  1. λi>0\lambda_i>0λi​>0 or i=1,2,⋯i=1,2,\cdotsi=1,2,⋯, F(x)F(x)F(x) have a single strong minimum
  2. λi<0\lambda_i<0λi​<0 or i=1,2,⋯i=1,2,\cdotsi=1,2,⋯, F(x)F(x)F(x) have a single strong maximum
  3. λi\lambda_iλi​ have both negative and positive together. F(x)F(x)F(x) has a saddle point
  4. λi≥0\lambda_i\geq 0λi​≥0 and have a λj=0\lambda_j=0λj​=0, F(x)F(x)F(x) has a weak minimum or has no stationary point.
