(draft)Action-value methods

Posted on 2020-04-06 Edited on 2020-11-08 In category Disqus:

Abstract: Keywords: value of action, greedy action, \(\varepsilon\)-greedy action, value function ## Estimating the Value of Action1 we have discussed the '\(k\)-armed bandit problem in the post 'An Introduction to \(k\)-armed Bandit Problem'. And now, we need to solve this problem with the information we have known. Our purpose in reinforcement learning is to maximize our cumulation of reward by selecting an action. However, the process does not stop for most steps except the last one, which means we can not know the final sum of reward as a certain action was selected in the middle of the process. So we want to find out the relation between the final sum of rewards and current actions which are not the last step.

The value function as we have described in the post presents the expectation of the accumulated rewards that an agent would get from current state \(s\) . And current state \(s\) is decided by our action in this \(k\)-armed bandit. So we can just build the relation between value and action. And the assumption in post 'An Introduction to \(k\)-armed Bandit Problem' defined that 'the reward of different actions have stationary probability distributions' which means the state would not change in the future step. So the expectation of sum of the reward in the future:

\[ \mathbb{E}[\sum_{t=\text{current}}^{\text{final}}R_t]=\sum_{t=\text{current}}^{\text{final}}\mathbb{E}[R_t]\tag{1} \]

that is under this condition that the environment is stationary which means the reward distribution do not change over time, the value of action \(a\) can be defined as:

\[ \text{value of action}\doteq \mathbb{E}[R_t|A_t = a]\tag{2} \]

To calculate the value of action in equation (2), we need the statistical tool which is called 'law of large number'. This law allow us to approximate the expectation by average. This gives that the value at the time \(t\) of action \(a\) is:

\[ Q_t(a)\doteq \frac{\sum_{1}^{t-1}R_i\mathbb{1}_{A_i=a}}{\sum_{1}^{t-1}\mathbb{1}_{A_i=a}}\tag{3} \]

when the denominator is \(0\), we can set the \(Q_t(a)\) be any number like \(0\). In the equation (3), \(\mathbb{1}_{A_i=a}\) is the indicator function that:

\[ \mathbb{1}_{A_i=a}= \begin{cases} 1 & \text{ if }A_i=a\\ 0 & \text{ otherwise } \end{cases}\tag{4} \]

Equation (3) just accumulate the rewards of the action that has been selected, so it is also called 'sample-average' method.

Selecting Actions Based on the Estimation

Once we got the action-value function, we can select the action based on the estimation. The first strategy is we can always select the greatest-value action, and this is called greedy action which exploits our current knowledge:

\[ A_t\doteq \mathrm{arg max}_aQ_t(a)\tag{5} \]

This strategy can bring us good rewards in a short time, but it can not sure the final sum of reward could be the best. This is because this algorithm spends no time to confirm that the current best action is the best or not. For example we have 3 actions in current step and the value of \(a_1\), \(a_2\) and \(a_3\) are \(1.0\), \(3.0\) and \(2.9\) respectively. \(a_2\) seems good but for this value are just average, we are not sure whether the expectation of \(a_3\) is smaller than the expectation of \(a_2\).

To overcome this drawback, we could exploit our knowledge at probability \(1-\varepsilon\) and exploration at probability \(\varepsilon\) at a single step. That is to say, we randomly select to greatest-value action with probability \(1-\varepsilon\)(exploitation) or other actions that do not have the greatest value with probability \(\varepsilon\)(exploration). This method is called \(\varepsilon\)-greedy method. When we explore, we select action randomly with an equal probability independent of the action-value estimation.

The law of large number make sure every \(Q_t(a)\) converge to \(q_{\star}(a)\) in probability as time \(t\) goes to infinity. And this also guarantees that the probability of selecting the greatest action converge to \(1-\varepsilon\)
