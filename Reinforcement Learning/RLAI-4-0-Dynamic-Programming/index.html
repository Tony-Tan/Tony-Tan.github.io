MPD has been introduced in previous posts:

And finite MDP is a more accessible subset of MDP who must have an optimal policy or a set of optimal policy who share the same value function of the same state as we hav discussed in the post ‘Optimal Policies and Optimal Value Functions’. And we did not give any algorithms to solve finite MDP task. Now, we start to investagate different kinds of algorithms to solve MDP problems. Dynamic programming(DP for short) is such a kind of algorithms which is not only easy to understand but essential to the following algorithms.

Dynamic Programming

Dynamic programming is a collect of algorithms, and they are employed to compute optimal policies given a perfect model of environment, i.e. the MDP condition. However, classical DP algorithms are limited utility in reinforcement learning because of its assumption of perfect model(MDP condition) and its great computational expense. But Dynamic programming is essential foundation for understanding of other methods which need less computation and are not based on strong assumption, like MDP condtion.

Finite MDP who contains finite

  1. stats,
  2. kinds of reward
  3. and actions

is the basis of dynamic programming. Dynamics who characterizes the environment is: $p(s’,r|s,a)$ where $s\in \mathcal{S}$, $a\in \mathcal{A}(s)$, $r\in \mathcal{R}$, and $s’\in \mathcal{S}^{+}$ and it has been introduced in the post ‘The Agent-Environment interface’. So if the dynamics are consistent, the environment is not changed from step to step as well. And what we need to concern is just the strategy of agent to find the optimal policies or the optimal value function.

Till now, all states and actions that we have talked about are all discrete and DP methods. In continuous state and action space, there must be exact solution to an MDP task. But we prefer to find an approximate solution of a quantilized state and action space for this is more effecient than searching in the continuous space.

Using value function of states pr state-action pairs is the key idea of DP methods and other reinforcement learning mehtods. It can be employed to organize and structure the search for good policies. And ‘Bellman optimality equation’ of states and state-action are used widely in DP methods.
