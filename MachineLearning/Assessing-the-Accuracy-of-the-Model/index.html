Assessing the Accuracy of the Model

Posted on 2019-10-23 Edited on 2020-11-08 In Artificial Intelligence , Machine Learning Disqus:

Keywords: linear regression

Accuracy of the Model[1]

An assumption about the model of the observed data is

Y=f(X)+ε(1)Y=f(X)+\varepsilon\tag{1} Y=f(X)+ε(1)

which means there is an actual generator behind the data and some noise was added during some phases. This is a general model for most regression problems. In linear regression mission, this would specialize as:

Y=w1X+w0+ε(2)Y=w_1X+w_0+\varepsilon\tag{2} Y=w1​X+w0​+ε(2)

In the post ‘simple linear regression problems’, we have mentioned a performance measurement of the linear model, and it is called ‘RSS\text{RSS}RSS’ (Residual Sum on Square). Because it has a strong connection with the performance of the linear model, which increases when the linear model fitted data worse. And it is used to direct the parameter searching in the learning phase. This kind of model performance, somehow, might be regarded as a way to define the accuracy of the model:

  • Lower RSS\text{RSS}RSS, better model
  • Higher RSS\text{RSS}RSS, worse model

Beside RSS\text{RSS}RSS, in this post, we take out another two numeral properties of the accuracy of the linear model.

  1. RSE\text{RSE}RSE
  2. R2\text{R}^2R2

Residual Standard Error(RSE\text{RSE}RSE)

RSS\text{RSS}RSS is a good tool to assess acuracy to model, but it also has deficiency. RSE\text{RSE}RSE is a derivative of RSS\text{RSS}RSS:

RSE=1n−2RSS=1n−2∑i=1n(yi−yi^)2(3)\text{RSE}=\sqrt{\frac{1}{n-2}\text{RSS}}=\sqrt{\frac{1}{n-2}\sum^n_{i=1}(y_i-\hat{y_i})^2}\tag{3} RSE=n−21​RSS​=n−21​i=1∑n​(yi​−yi​^​)2​(3)

Factor 222 in n−2n-2n−2 came from that there are 2 parameters in our simple linear regression model. And if there are mmm parameters in our model, this factor will be changed to n−mn-mn−m:

RSE=1n−mRSS=1n−m∑i=1n(yi−yi^)2(4)\text{RSE}=\sqrt{\frac{1}{n-m}\text{RSS}}=\sqrt{\frac{1}{n-m}\sum^n_{i=1}(y_i-\hat{y_i})^2}\tag{4} RSE=n−m1​RSS​=n−m1​i=1∑n​(yi​−yi​^​)2​(4)

Why RSE\text{RSE}RSE is a better choice than RSE\text{RSE}RSE?

One of the assumption of equation(1) is that ε\varepsilonε is random variable whose has Gaussian distribution with mean 000 . Then every YiY_iYi​ also has a Gaussian distribution with mean w1Xi+w0w_1X_i+w_0w1​Xi​+w0​ and variance δ2\delta^2δ2. In another world:

δi2=(yi^−yi)2\delta_i^2=(\hat{y_i}-y_i)^2 δi2​=(yi​^​−yi​)2

And it is the essential observation to estimate the real δ2\delta^2δ2. And all XiX_iXi​'s are i.i.d, which means they have the same δ2\delta^2δ2. And RSS\text{RSS}RSS can be a statistic to predict δ2\delta^2δ2. To make an unbias estimation of standard deviation δ\deltaδ, RSS\text{RSS}RSS is converted to RSE\text{RSE}RSE.

In other words, when we have the actual parameters, RSE\text{RSE}RSE can be used as an unbias estimate of the standard deviation of δ\deltaδ in equation (1) (δi2=(yi^−yi)2\delta_i^2=(\hat{y_i}-y_i)^2δi2​=(yi​^​−yi​)2 has a χ2\chi^2χ2 distribution). But, when we don’t have the correct parameters, RSE\text{RSE}RSE has the same effect of RSS\text{RSS}RSS.

RSE\text{RSE}RSE measures the lake of fitting:

  • when yi^∼yi\hat{y_i}\sim y_iyi​^​∼yi​, we have a small RSE\text{RSE}RSE – good fit
  • when yi^ far from yi\hat{y_i} \text{ far from } y_iyi​^​ far from yi​, we have a big RSE\text{RSE}RSE – bad fit

R2\text{R}^2R2

Consider the following situation: if the value of XiX_iXi​ is among [109,1010][10^9,10^{10}][109,1010], the RSE\text{RSE}RSE of our model may be bigger than 1 million. And another model of the same data, but using the conversion, like log⁡(Xi)\log(X_i)log(Xi​), the RSE\text{RSE}RSE, of the new model may lie below 100. Then which model is good can not be told just by RSE\text{RSE}RSE. We need a relative measure but not an absolute measure.

A traditional way of changing absolute measure to a relative one is building a propotion. So we get:

R2=TSS−RSSTSS=1−RSSTSS\text{R}^2=\frac{\text{TSS}-\text{RSS}}{\text{TSS}}=1-\frac{\text{RSS}}{\text{TSS}} R2=TSSTSS−RSS​=1−TSSRSS​

This gives us the following conclusion:

  1. 0≤R2≤10 \leq \text{R}^2 \leq 10≤R2≤1
  2. When R2=1\text{R}^2 = 1R2=1, RSS=0\text{RSS}=0RSS=0, that means a perfect fitting and the model explain everything
  3. When R2=0\text{R}^2 = 0R2=0, RSS=TSS\text{RSS}=\text{TSS}RSS=TSS, this means the fitting does nothing for predicting, because its prediction is just the same as the mean of the sample.

So, R2\text{R}^2R2 is a good measurement for the accuracy of the model.

Usage of R2\text{R}^2R2

Usage of R2\text{R}^2R2 is not just an indicator of how good a model is, but it can be used in any way. The first one, for instance, in Physics, we know X,YX, YX,Y has a linear relationship, but R2\text{R}^2R2 after regression is close to 0. So, this R2\text{R}^2R2 gives information that data has too much noise, or even the experiment is totally wrong.

The second example is using R2\text{R}^2R2 to test whether the sample has a linear relationship. What we need to do is do linear regression and then test its R2\text{R}^2R2, if R2→0\text{R}^2\to 0R2→0, which means they are not linear.

Cov(X,Y)\text{Cov}(X,Y)Cov(X,Y) and R2\text{R}^2R2

We test if the linear relationship is strong by R2\text{R}^2R2, this is a little like we test if there was some relationship between two variables by covariance. And R2\text{R}^2R2 is a linear transform of covariance of XXX and YYY. However we use R2\text{R}^2R2 instead of Cov(X,Y)\text{Cov}(X,Y)Cov(X,Y) because in multi-variables case R2\text{R}^2R2 works easier.
