K-means Clustering

Posted on 2020-03-04 Edited on 2020-11-08 In Artificial Intelligence , Machine Learning Disqus:

Keywords: K-means

Original form K-Means algorithm might be one of the most accessible algorithms in machine learning. And many books and courses started with it. However, if we convert the task which K-means dealt with into a more mathematical form, there would be more interesting aspects coming to us.

Clustering Problem[1]

The first thing we should do before introducing the algorithm is to make the task clear. And a precise mathematical form is usually the best way.

Clustering is a kind of unsupervised learning task. So there is no correct nor incorrect solution. Clustering is similar to classification during predicting since the output of clustering and classification is discrete. However, during training classifiers, we always have a certain target corresponding to every input. On the contrary, clustering has no target at all, and what we have is only

{x1,⋯ ,xN}(1)\{x_1,\cdots, x_N\}\tag{1} {x1​,⋯,xN​}(1)

where xi∈RDx_i\in\mathbb{R}^Dxi​∈RD for i=1,2,⋯ ,Ni=1,2,\cdots,Ni=1,2,⋯,N. And our mission is to separate the dataset into KKK groups (where KKK has been given before task)

An intuitive strategy of clustering based on two considerations:

  1. the distance between data points in the same group should be as small as possible.
  2. the distance between data points in the different groups should be as large as possible.

Basing on these two points, some concepts were formed. The first one is how to represent a group. We take

μi:i∈{1,2,⋯ ,K}(2)\mu_i:i\in\{1,2,\cdots, K\}\tag{2} μi​:i∈{1,2,⋯,K}(2)

as the prototype associated with iiith group. A group always contains several points, and a spontaneous idea is using the center of all the points belonging to the group as its prototype. To represent which group xi\boldsymbol{x}_ixi​ in equation (1) belongs to, an indicator is necessary, and 1-of-K coding scheme is used, where the indicator:

rnk∈{0,1}(3)r_{nk}\in\{0,1\}\tag{3} rnk​∈{0,1}(3)

for k=1,2,⋯ ,Kk=1,2,\cdots,Kk=1,2,⋯,K representing the group number and n=1,2,⋯ ,Nn = 1,2,\cdots,Nn=1,2,⋯,N denoting the number of sample point, and where rnk=1r_{nk}=1rnk​=1 then rnj=0r_{nj}=0rnj​=0 for all j≠kj\neq kj​=k.

Objective Function

A loss function is a good way to measure the quantity of our model during both training and testing stages. And in the clustering task loss function could not be used for we have no idea about loss. However, we can build another function that plays the same role as loss function and it is also the target of what we want.

According to the two base points above, we build our objective function:

J=∑n=1N∑k=1Krnk∣∣xn−μk∣∣2(4)J=\sum_{n=1}^{N}\sum_{k=1}^{K}r_{nk}||\boldsymbol{x}_n-\mu_k||^2\tag{4} J=n=1∑N​k=1∑K​rnk​∣∣xn​−μk​∣∣2(4)

In this objective function, the distance is defined as Euclidean distance(However, other measurements of similarity could also be used). Then the mission is to minimize JJJ by finding some certain {rnk}\{r_{nk}\}{rnk​} and {μk}\{\mu_k\}{μk​}

K-Means Algorithm

Now, let’s represent the famous K-Means algorithm. The method includes two steps:

  1. Minimising JJJ respect to rnkr_{nk}rnk​ keeping μk\mu_kμk​ fixed
  2. Minimising JJJ respect to μk\mu_kμk​ keeping rnkr_{nk}rnk​ fixed

In the first step, according to equation (4), the objective function is linear of rnkr_{nk}rnk​. So there is a close solution. Then we set:

rnk={1 if k=argminj∣∣xn−μj∣∣20otherwise(5)r_{nk}=\begin{cases} 1&\text{ if } k=\mathop{argmin}_{j}||x_n-\mu_j||^2\\ 0&\text{otherwise} \end{cases}\tag{5} rnk​={10​ if k=argminj​∣∣xn​−μj​∣∣2otherwise​(5)

And in the second step, rnkr_{nk}rnk​ is fixed and we minimise objective function JJJ. For it is quadratic, the minimume point is on the stationary point where:

∂J∂μk=−∑n=1Nrnk(xn−μk)=0(6)\frac{\partial J}{\partial \mu_k}=-\sum_{n=1}^{N}r_{nk}(x_n-\mu_k)=0\tag{6} ∂μk​∂J​=−n=1∑N​rnk​(xn​−μk​)=0(6)

and we get:

μk=∑n=1Nrnkxn∑n=1Nrnk(7)\mu_k = \frac{\sum_{n=1}^{N}r_{nk}x_n}{\sum_{n=1}^{N}r_{nk}}\tag{7} μk​=∑n=1N​rnk​∑n=1N​rnk​xn​​(7)

rnkr_{nk}rnk​ is the total number of points from the sample {x1,⋯ ,xN}\{x_1,\cdots, x_N\}{x1​,⋯,xN​} who belong to prototype μk\mu_kμk​ or group kkk at current step. And μk\mu_kμk​ is just the average of all the points in the group kkk.

This two-step, which was calculated by equation (5),(7), would repeat until rnkr_{nk}rnk​ and μk\mu_kμk​ not change.

The K-means algorithm guarantees to converge because at every step the objective function JJJ is reduced. So when there is only one minimum, the global minimum, the algorithm must converge.

Input Data Preprocessing before K-means

Most algorithms need their input data to obey some rules. To the K-means algorithm, we rescale the input data to mean 0 and variance 1. This is always done by
