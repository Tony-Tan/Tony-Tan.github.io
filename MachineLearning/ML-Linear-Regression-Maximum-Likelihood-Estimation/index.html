Square Loss Function for Regression[1]

To any input x\boldsymbol{x}x, our goal in a regression task is to give a prediction y^=y(x)\hat{y}=y(\boldsymbol{x})y^​=y(x) to approximate target ttt where the function yyy is the chosen hypothesis. And the difference between ttt and y^\hat{y}y^​ can be called ‘error’ or more precisely ‘loss’. Because in an approximation task, ‘error’ occures by chance and alway exists, but ‘loss’ is a good word to describe the difference. The loss can be written generally as function ℓ(y(x),t)\ell(y(\boldsymbol{x}),t)ℓ(y(x),t). Intuitively, the smaller the loss, the better the approximation. So the expection of loss:

E[ℓ]=∫∫ℓ(y(x),t)p(x,t)dxdt(1)\mathbb E[\ell]=\int\int \ell(y(\boldsymbol{x}),t)p(\boldsymbol{x},t)d \boldsymbol{x}dt\tag{1} E[ℓ]=∫∫ℓ(y(x),t)p(x,t)dxdt(1)

should be small.

In probability viewpoint, the input vector x\boldsymbol{x}x, target ttt and parameters in function yyy are all random variables. So the expectation of loss function exists.

Considering the sqare error loss function e=(y(x)−t)2e=(y(\boldsymbol{x})-t)^2e=(y(x)−t)2, it is a way to present the difference between the prediction of our prediction and the target. And substitute the loss funtion into equation (1), we have:

E[ℓ]=∫∫(y(x)−t)2p(x,t)dxdt(2)\mathbb E[\ell]=\int\int (y(\boldsymbol{x})-t)^2p(\boldsymbol{x},t)d \boldsymbol{x}dt\tag{2} E[ℓ]=∫∫(y(x)−t)2p(x,t)dxdt(2)

To minimize this function, we could use Euler-Lagrange equation, Fundamental theorem of calculus and Fubini’s theorem:

Fubini’s theorem told us that we can change the order of intergration:

E[ℓ]=∫∫(y(x)−t)2p(x,t)dxdt=∫∫(y(x)−t)2p(x,t)dtdx(3)\begin{aligned} \mathbb E[\ell]&=\int\int (y(\boldsymbol{x})-t)^2p(\boldsymbol{x},t)d \boldsymbol{x}dt\\ &=\int\int (y(\boldsymbol{x})-t)^2p(\boldsymbol{x},t)dtd \boldsymbol{x} \end{aligned}\tag{3} E[ℓ]​=∫∫(y(x)−t)2p(x,t)dxdt=∫∫(y(x)−t)2p(x,t)dtdx​(3)

According to the Euler-Lagrange equation, we first create a new function G(x,y,y′)G(x,y,y')G(x,y,y′):

G(x,y,y′)=∫(y(x)−t)2p(x,t)dt(4)G(x,y,y')= \int (y(\boldsymbol{x})-t)^2p(\boldsymbol{x},t)dt\tag{4} G(x,y,y′)=∫(y(x)−t)2p(x,t)dt(4)

Euler-Lagrange equation is used to minimize the equation (2):

∂G∂y−ddx∂G∂y′=0(5)\frac{\partial G}{\partial y}-\frac{d}{dx}\frac{\partial G}{\partial y'}=0\tag{5} ∂y∂G​−dxd​∂y′∂G​=0(5)

Because there is no y′y'y′ component in function G()G()G(). Then the equation:

∂G∂y=0(6)\frac{\partial G}{\partial y}=0\tag{6} ∂y∂G​=0(6)

becomes the necessary condition to minimize the equation (2):

2∫(y(x)−t)p(x,t)dt=0(7)2\int (y(\boldsymbol{x})-t)p(\boldsymbol{x},t)dt=0 \tag{7} 2∫(y(x)−t)p(x,t)dt=0(7)

What we want to find to minimize the loss is predictor yyy, so rearrange the equation (6), and we get a good predictor that can minimize the square loss function :

∫(y(x)−t)p(x,t)dt=0∫y(x)p(x,t)dt−∫tp(x,t)dt=0y(x)∫p(x,t)dt=∫tp(x,t)dty(x)=∫tp(x,t)dt∫p(x,t)dty(x)=∫tp(x,t)dtp(x)y(x)=∫tp(t∣x)dty(x)=Et[t∣x](8)\begin{aligned} \int (y(\boldsymbol{x})-t)p(\boldsymbol{x},t)dt&=0\\ \int y(\boldsymbol{x})p(\boldsymbol{x},t)dt-\int tp(\boldsymbol{x},t)dt&=0\\ y(\boldsymbol{x})\int p(\boldsymbol{x},t)dt&=\int tp(\boldsymbol{x},t)dt\\ y(\boldsymbol{x})&=\frac{\int tp(\boldsymbol{x},t)dt}{\int p(\boldsymbol{x},t)dt}\\ y(\boldsymbol{x})&=\frac{\int tp(\boldsymbol{x},t)dt}{p(\boldsymbol{x})}\\ y(\boldsymbol{x})&=\int tp(t|\boldsymbol{x})dt\\ y(\boldsymbol{x})&= \mathbb{E}_t[t|\boldsymbol{x}] \end{aligned}\tag{8} ∫(y(x)−t)p(x,t)dt∫y(x)p(x,t)dt−∫tp(x,t)dty(x)∫p(x,t)dty(x)y(x)y(x)y(x)​=0=0=∫tp(x,t)dt=∫p(x,t)dt∫tp(x,t)dt​=p(x)∫tp(x,t)dt​=∫tp(t∣x)dt=Et​[t∣x]​(8)

To minimize the expection of the square loss function, we finally find the expection of ttt given x\boldsymbol{x}x is the optimum solution to the task which means the solution of y(x)=E[t∣x]y(\boldsymbol{x})=\mathbb{E[t| \boldsymbol{x}]}y(x)=E[t∣x]. The expection of ttt given x\boldsymbol{x}x is also called the regression function.

A small summary: E[t∣x]\mathbb{E[t| \boldsymbol{x}]}E[t∣x] is a good expection of y(x)y(\boldsymbol{x})y(x)

Maximum Likelihood Estimation

Generally, we assume that there is a generator behand the data:

t=g(x,w)+ϵ(9)t=g(\boldsymbol{x},\boldsymbol{w})+\epsilon\tag{9} t=g(x,w)+ϵ(9)

where the function g(x,w)g(\boldsymbol{x},\boldsymbol{w})g(x,w) is a deterministic function, ttt is the target variable and ϵ\epsilonϵ is zero mean Gaussian random variable with percision β\betaβ which is the inverse variance. Because of the property of Gaussian distribution, ttt has a Gaussian distribution, with mean(expectation) g(x,w)g(\boldsymbol{x},\boldsymbol{w})g(x,w) and percesion β\betaβ. And recalling the standard form of Gaussian distribution:

P(t∣x,w,β)=N(t∣g(x,w),β−1)=β2πe−12(β(x−μ)2)(10)\begin{aligned} \mathbb{P}(t|\boldsymbol{x},\boldsymbol{w},\beta)&=\mathcal{N}(t|g(\boldsymbol{x},\boldsymbol{w}),\beta^{-1})\\ &=\frac{\beta}{\sqrt{2\pi}}\mathrm{e}^{-\frac{1}{2}(\beta(x-\mu)^2)} \end{aligned}\tag{10} P(t∣x,w,β)​=N(t∣g(x,w),β−1)=2π​β​e−21​(β(x−μ)2)​(10)

What our task here is to approximate the generator in equation (9) with a linear function. Somehow, when we use the square loss function, the optimum solution for this task is E[t∣x]\mathbb{E}[t|\boldsymbol{x}]E[t∣x] with respect of equation (8). And to the equation (10) the solution is:

E[t∣x]=g(x,w)(11)\mathbb{E}[t|\boldsymbol{x}]=g(\boldsymbol{x},\boldsymbol{w})\tag{11} E[t∣x]=g(x,w)(11)

Then we set our linear model as:

y(x)=wTx+b(12)y(x)=\boldsymbol{w}^T\boldsymbol{x}+b\tag{12} y(x)=wTx+b(12)

and this can be transformed as:

y(x)=[bwT][1x]=waTxa(13)y(x)= \begin{bmatrix} b&\boldsymbol{w}^T \end{bmatrix} \begin{bmatrix} 1\\ \boldsymbol{x} \end{bmatrix}=\boldsymbol{w}_a^T\boldsymbol{x}_a \tag{13} y(x)=[b​wT​][1x​]=waT​xa​(13)

for short, we just write the wa\boldsymbol{w}_awa​ and xa\boldsymbol{x}_axa​ as w\boldsymbol{w}w and x\boldsymbol{x}x. Then the linear model becomes:

y(x)=wTx(14)y(x)=\boldsymbol{w}^T\boldsymbol{x}\tag{14} y(x)=wTx(14)

and to get the prediction we need to find out what w\boldsymbol{w}w is. As we mentioned above we consider all the parameter as a random variable, then the conditioned distribution of w\boldsymbol{w}w is P(w∣t,β)\mathbb{P}(\boldsymbol{w}|\boldsymbol{t},\beta)P(w∣t,β). XXX or x\boldsymbol{x}x could be omitted for its distribution will not be concerned. And the Bayesian theorem told us:

P(w∣t,β)=P(t∣w,β)P(w)P(t)=Likelihood×PriorEvidence(15)\mathbb{P}(\boldsymbol{w}|\boldsymbol{t},\beta)=\frac{\mathbb{P}( \boldsymbol{t}|\boldsymbol{w},\beta) \mathbb{P}(\boldsymbol{w})} {\mathbb{P}(\boldsymbol{t})}=\frac{\text{Likelihood}\times \text{Prior}}{\text{Evidence}}\tag{15} P(w∣t,β)=P(t)P(t∣w,β)P(w)​=EvidenceLikelihood×Prior​(15)

Because P(t)\mathbb{P}(\boldsymbol{t})P(t) and P(w)\mathbb{P}(\boldsymbol{w})P(w) are constant. Then the maximum of likelihood P(t∣w,β)\mathbb{P}(\boldsymbol{t}|\boldsymbol{w},\beta)P(t∣w,β) maximise the posterior probability.

P(t∣w,β)=Πi=0NN(ti∣wTxi,β−1)ln⁡P(t∣w,β)=∑i=0Nln⁡N(ti∣wTxi,β−1)=∑i=0Nln⁡β2πe−12(β(ti−wTxi)2)=∑i=0Nln⁡β−∑i=0Nln⁡2π−12β∑i=0N(ti−wTxi)2(16)\begin{aligned} \mathbb{P}(\boldsymbol{t}|\boldsymbol{w},\beta)&=\Pi_{i=0}^{N}\mathcal{N}(t_i|\boldsymbol{w}^T\boldsymbol{x}_i,\beta^{-1})\\ \ln \mathbb{P}(\boldsymbol{t}|\boldsymbol{w},\beta)&=\sum_{i=0}^{N}\ln \mathcal{N}(t_i|\boldsymbol{w}^T\boldsymbol{x}_i,\beta^{-1})\\ &=\sum_{i=0}^{N}\ln \frac{\beta}{\sqrt{2\pi}}\mathrm{e}^{-\frac{1}{2}(\beta(t_i-\boldsymbol{w}^T\boldsymbol{x}_i)^2)}\\ &=\sum_{i=0}^{N} \ln \beta - \sum_{i=0}^{N} \ln \sqrt{2\pi} - \frac{1}{2}\beta\sum_{i=0}^{N}(t_i-\boldsymbol{w}^T\boldsymbol{x}_i)^2 \end{aligned}\tag{16} P(t∣w,β)lnP(t∣w,β)​=Πi=0N​N(ti​∣wTxi​,β−1)=i=0∑N​lnN(ti​∣wTxi​,β−1)=i=0∑N​ln2π​β​e−21​(β(ti​−wTxi​)2)=i=0∑N​lnβ−i=0∑N​ln2π​−21​βi=0∑N​(ti​−wTxi​)2​(16)

Least Square Estimation and Maximum Likelihood Estimation

When we assume there is a generator:

t=g(x,w)+ϵ(18)t=g(\boldsymbol{x},\boldsymbol{w})+\epsilon\tag{18} t=g(x,w)+ϵ(18)

behind the data, and ϵ\epsilonϵ has a zero-mean Gaussian distribution with any precision β\betaβ, the maximum likelihood estimation finally converts to the least square estimation. This is not only worked for linear regression, because we have no assumption about what g(x,w)g(\boldsymbol{x},\boldsymbol{w})g(x,w) is.

However, when the ϵ\epsilonϵ has a different distribution but not Gaussian distribution, the least square estimation will not be the optimum solution.
