Analysis of Committees[1]

The committee is a native inspiration for how to combine several models(or we can say how to combine the outputs of several models). For example, we can combine all the models by:

yCOM(X)=1M∑m=1Mym(X)(1)y_{COM}(X)=\frac{1}{M}\sum_{m=1}^My_m(X)\tag{1} yCOM​(X)=M1​m=1∑M​ym​(X)(1)

However, we want to analyze whether this average prediction of models is good than a single one of them.

To compare the committee and a single model, we should first build a criterion depending on which we can distinguish which model is better. Assuming that the true generator of the training data xxx is:

h(x)(2)h(x)\tag{2} h(x)(2)

So our prediction of the mmmth model for m=1,2⋯ ,Mm=1,2\cdots,Mm=1,2⋯,M can be represented as:

ym(x)=h(x)+ϵm(x)(3)y_m(x) = h(x) +\epsilon_m(x)\tag{3} ym​(x)=h(x)+ϵm​(x)(3)

then the average sum-of-square of error can be a nice criterion. And the criterion of sigle model is:

Ex[(ym(x)−h(x))2]=Ex[ϵm(x)2](4)\mathbb{E}_x[(y_m(x)-h(x))^2] = \mathbb{E}_x[\epsilon_m(x)^2] \tag{4} Ex​[(ym​(x)−h(x))2]=Ex​[ϵm​(x)2](4)

where the E[⋅]\mathbb{E}[\cdot]E[⋅] is the frequentist expectation. To make the creterion more concrete, we consider the average of the error over MMM models:

EAV=1M∑m=1MEx[ϵm(x)2](5)E_{AV} = \frac{1}{M}\sum_{m=1}^M\mathbb{E}_x[\epsilon_m(x)^2]\tag{5} EAV​=M1​m=1∑M​Ex​[ϵm​(x)2](5)

And on the other hand, the committees have the error given by equation(1), (3) and (4):

ECOM=Ex[(1M∑m=1Mym(x)−h(x))2]=Ex[{1M∑m=1Mϵm(x)}2](6)\begin{aligned} E_{COM}&=\mathbb{E}_x[(\frac{1}{M}\sum_{m=1}^My_m(x)-h(x))^2] \\ &=\mathbb{E}_x[\{\frac{1}{M}\sum_{m=1}^M\epsilon_m(x)\}^2] \end{aligned} \tag{6} ECOM​​=Ex​[(M1​m=1∑M​ym​(x)−h(x))2]=Ex​[{M1​m=1∑M​ϵm​(x)}2]​(6)

Now we assume that the random variables ϵi(x)\epsilon_i(x)ϵi​(x) for i=1,2,⋯ ,Mi=1,2,\cdots,Mi=1,2,⋯,M have mean 0 and uncorrelated, so that:

Ex[ϵm(x)]=0Ex[ϵm(x)ϵl(x)]=0,m≠l(7)\begin{aligned} \mathbb{E}_x[\epsilon_m(x)]&=0 &\\ \mathbb{E}_x[\epsilon_m(x)\epsilon_l(x)]&=0,&m\neq l \end{aligned} \tag{7} Ex​[ϵm​(x)]Ex​[ϵm​(x)ϵl​(x)]​=0=0,​m​=l​(7)

Then subsitute equ (7) into equ (6), we can get:

ECOM=1M2Ex[ϵm(x)](8)E_{COM}=\frac{1}{M^2}\mathbb{E}_x[\epsilon_m(x)]\tag{8} ECOM​=M21​Ex​[ϵm​(x)](8)

According the equation (5) and (8):

EAV=1MECOM(9)E_{AV}=\frac{1}{M}E_{COM}\tag{9} EAV​=M1​ECOM​(9)

All the mathematics above is based on the assumption that the error of each model is uncorrelated. However, most time they are highly correlated and the reduction of error is generally small. But the relation:

ECOM≤EAV(10)E_{COM}\leq E_{AV}\tag{10} ECOM​≤EAV​(10)

exists for sure. Then the boosting method was established to make the combining model more powerful.
